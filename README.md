# dotfiles

### プラグイン追加
`zsh/plugins.zsh`に追加していく。  
追加後、`source ~/.zshrc`。

### fast-syntax-highlightingのテーマ変更
`fast-theme ~/dotfiles/zsh/.config/catppuccin-frappe.ini`コマンドで適用
